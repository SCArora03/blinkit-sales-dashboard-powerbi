# Blinkit Sales Dashboard (Power BI)

## Overview
An interactive Power BI dashboard analyzing Blinkit's retail sales 
performance across outlet types, locations, item categories, and fat 
content — built to surface actionable insights from 8,523 product entries 
totaling $1.20M in sales.

## Tools Used
- Power BI (KPI Cards, Donut Charts, Bar Charts, Line Charts, 
  Slicers, Data Tables)

## Dataset Source
 BlinkIT Grocery Data.xlsx

## Key KPIs
- Total Sales: $1.20M
- Average Sales: $141
- No. of Items: 8,523
- Average Rating: 3.9

## Key Insights
- Tier 3 outlets generate the highest sales despite being smaller 
  markets — suggesting strong demand outside metro cities
- Low Fat products outsell Regular products, reflecting growing 
  health consciousness among customers
- Sales peaked in 2018 at $205K, declining slightly in subsequent 
  years — indicating possible market saturation
- Fruits & Vegetables is the top-performing item category
- Supermarket Type 1 dominates with $787.55K in sales — nearly 5x 
  more than any other outlet type

## Filters Available
- Outlet Location Type (Tier 1 / 2 / 3)
- Outlet Size (Small / Medium / High)
- Item Type
